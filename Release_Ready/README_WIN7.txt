KSB 初始性能报告生成工具 (Win7 兼容版)
========================================

使用方法：
1. 解压 Release_Ready.zip
2. 双击 KsbReportTool.exe 运行
3. 在界面中选择两个输入文件：
   - KSB原始报告汇总表.xlsx (表1)
   - KSB编号对照汇总表.xlsx (表2)
   - 模板已内置（OH2），无需选择
4. 点击"开始生成"按钮
5. 生成的报告将保存在输出目录中

系统要求：
- Windows 7 SP1 或更高版本
- .NET Framework 4.8 (Win7 需单独安装)

文件说明：
- KsbReportTool.exe - 主程序
- KsbReportTool.exe.config - 配置文件
- *.dll - 运行依赖库 (NPOI, SharpZipLib 等)
